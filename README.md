# WatchX - Premium Stopwatch Web App

A modern, interactive stopwatch application built with vanilla HTML, CSS, and JavaScript. Features millisecond precision, lap tracking, analytics, and customizable backgrounds.


## ✨ Features

| Feature | Description |
| --- | --- |
| **Millisecond Precision** | Smooth 60fps timer using `Date.now()` for accuracy |
| **Start / Pause / Reset** | Full stopwatch controls with state management |
| **Lap Tracking** | Record unlimited laps with individual lap times + total time |
| **Lap Analytics** | Auto-highlights fastest lap in green, slowest in red. Shows average |
| **Export Data** | Download lap history as CSV file for analysis |
| **Dark / Light Mode** | Toggle themes with preference saved to localStorage |
| **Custom Backgrounds** | Choose from presets or upload your own wallpaper |
| **Keyboard Shortcuts** | `Space` = Start/Stop, `L` = Lap, `R` = Reset, `Esc` = Close modals |
| **Sound Feedback** | Subtle audio cues on start, stop, and lap actions |
| **Responsive Design** | Glassmorphism UI that works on desktop and mobile |
| **Zero Dependencies** | Pure vanilla JS. No frameworks, no build step |

## 🚀 Demo

Just open `index.html` in any modern browser. No installation required.
